Política de Privacidad — Frikeys Café Restaurante Skill

Última actualización: junio 2026

1. Información que recopilamos

La Skill de Frikeys Café Restaurante recopila y almacena únicamente la siguiente información proporcionada voluntariamente por el usuario:

  • Nombre de usuario (alias)
  • Contraseña (almacenada en base de datos segura Aiven PostgreSQL con conexión SSL)
  • Dirección de entrega proporcionada al realizar un pedido
  • Historial de pedidos realizados durante la sesión

2. Cómo usamos la información

La información recopilada se utiliza exclusivamente para:

  • Autenticar al usuario dentro de la skill
  • Procesar y gestionar los pedidos realizados
  • Mostrar el menú y opciones personalizadas del restaurante

3. Almacenamiento y seguridad

Los datos del usuario se almacenan en una base de datos PostgreSQL alojada en Aiven Cloud con conexión SSL. No compartimos, vendemos ni transferimos información personal a terceros.

4. Datos de menores de edad

Esta skill no está dirigida a menores de edad. No recopilamos intencionalmente información de menores de 13 años.

5. Retención de datos

Los datos de sesión (pedidos, ubicación) se eliminan al finalizar la sesión de la skill. Los datos de cuenta (usuario y contraseña) se conservan hasta que el usuario solicite su eliminación.

6. Derechos del usuario

El usuario puede solicitar la eliminación de su cuenta y datos asociados contactando a: frikeys.cafe@uthh.edu.mx
